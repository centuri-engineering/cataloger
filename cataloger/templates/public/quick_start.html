Cards and Annotations

This service helps you create and use cards that correspond to a project, an experiment or a set of experiments. Each card has various annotations for different categories:

  • Organism: fruit fly, Xenopus, human...
  • Sample: a cell line, an organ, a tissue, an organelle ...
  • Process: mesoderm invagination, mitosis, aging ...
  • Method: laser ablation, FISH, RNAi ...
  • Gene: alpha-tubulin, ROCK, Myosin II ...

Note that those categories are loosely defined, but each annotation will be tied to an entry in one of bioportal ontologies, so the vocabulary will be unequivocal
